# EShop
EShop is an e-commerce app that allows users to browse, purchase, and sell products. It is built using Cloud Firestore and developed using Kotlin.

# Features
- User authentication: Securely authenticate users using Firebase Authentication.
- Dashboard: A single view of all products available for purchase, as well as orders placed and sold.
- Product Section: A dedicated section for adding and managing products for sale.
- Orders Section: A list of all orders placed and sold, with the ability to track their status and manage returns.
- Cart Section: A shopping cart for adding and managing items to purchase.
- Settings Section: A section for changing user profile information and other settings.

# Technologies Used
- Firebase Authentication: For user authentication and authorization.
- Android Studio: The app is developed using Android Studio.
- Cloud Firestore: For storing and managing user data and posts. 
- Kotlin: The primary programming language used for app development.
